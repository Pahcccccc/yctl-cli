## [0.1.0] - 2026-01-14

### Added
- Initial release of `yctl` CLI tool.
- Project initialization templates (`nlp`, `cv`, `ml`, `research`).
- Dataset inspection for CSV, Excel, Parquet, and JSON.
- System diagnostics with `yctl doctor`.
- AI project idea analysis with `yctl think`.
- Comprehensive testing suite (100% core logic coverage).
- Professional `README.md` and `CONTRIBUTING.md`.
